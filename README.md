# MailMan
A Python GUI for making GET, POST, PUT and DELETE requests

## Goal
make a GUI for making requests under 24 hours

## Features
- GET requests
- POST requests
- PUT requests
- DELETE requests
- Custom headers
- Custom Body
- Body type: JSON, FormData, PlainText
- Save requests
- Load requests
- Response Display
- Logging
